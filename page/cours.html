Курс по Front-End Development

1-й месяц Front-End Core:

Верстка сайтов с помощью HTML5, CSS3 и Bootstrap, эффективное пользование MDB и Tailwind CSS, инструменты Webpack/Npm/Yarn instrumentlari и др.;

2-ой месяц Front-End Kernel:

JavaScript (OOP/ES6-9) язык программирования, библиотеки jQuery/Vanilla, Ajax, SSE, эффективное пользование Websockets, технологии PWA;

3-й месяц Front-End Standart:

Создание целевых страниц с помощью Vue, Vue CLI, Vue DevTools, Vuex, Axios, Storybook, Vue-Router, SSR, Quasar Framework и других эффективных библиотек;

4-й месяц Front-End Advanced:

Разработка мобильных приложений для Android и iOS систем с помощью Ionic2/Ionic Native/IonicCLI и Capacitor.js

Курс по Back-End Development

1-й месяц Back-End Core:

Основы PHPStorm IDE, Nginx, PHP7 и OOP. Эффективное пользование Composer/PECL, база данных Navicat, PostgreSQL и MongoDB, пользование XDebug. Работа с помощью CLI/CGI/FastCGI(FPM);

2-ой месяц Back-End Kernel:

Усложненный курс по PHP7 и OOP, основы платформ ZET Framework (Laravel/Symfony/Yii2), работа с помоRoutes, Database, Active Record ORM, Models, Widgets, Session/Cookies/LocalStorage;

3-й месяц Back-End Standart:

Усложненный курс по ZET Framework, использование сервисов Services, Actions, Events, Console & Web Apps и IoC/DI, эффективная работа с помощью Widgets

4-й месяц Back-End Advanced:

Performance tuning (Opcache, Redis, ElasticSearch, BlackFire), Secure Coding, XSS, SQL/PHP/Shell Injections Prevention, CGI/ Filesystem/Database security, Nginx, MongoDB, PHPCI, CircleCI, REST API, Selenium, разработка Telegram.Bot (Botman);
